# Mawrid Frontend (Alpha)

Frontend application for the Mawrid AI knowledge platform.

## Overview

Mawrid Frontend provides the user interface for interacting with the Mawrid platform.

It allows users to:

- Manage knowledge bases
- Upload and organize documents
- Monitor document processing
- Search and interact with AI-powered knowledge

The frontend communicates with the Mawrid Backend API to manage data and trigger AI processing workflows.
